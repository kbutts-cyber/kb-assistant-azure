# KB Assistant

KB Assistant is an AI-powered knowledge base tool built on Azure.
It uses Azure OpenAI GPT-4o-mini to answer questions about documents.

## How to use
Type your password and ask a question in plain English.
The assistant will search the knowledge base and return an answer.

## Technology Stack
- Azure Container Apps
- Azure OpenAI GPT-4o-mini
- Azure Blob Storage
- Azure Key Vault
- Terraform
- GitHub Actions
- Docker
- FastAPI
